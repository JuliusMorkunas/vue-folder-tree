# vue-folder-tree

[![License][license-badge]][license-url]

> Simple folder tree written in Vue using Vuetify components and Vuex store.

## Project setup

```
npm install
```

### Compile and hot-reload for development

```
npm start
```

### Run unit tests

```
npm test
```

### Compile and minify for production

```
npm run build
```

### Lint files

```
npm run lint
```

### Format files

```
npm run prettier
```
